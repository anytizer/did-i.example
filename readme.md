# Did I?

A demonstration purpose list manager.


## Implemenation

 * Cross-Controller communication
 * Angular Service
 * API Integration, HTTP Requests
 * SQLite Database
 * Soft Deletes
 * User Interface
 * Object oriented programming
 * HTML Presentation
 * Scripts protection
